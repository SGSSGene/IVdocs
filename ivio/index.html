Introduction

A c++20 library for bioinformatics. It provides functions to load and save typical file formats.

Basic concepts

Each file type consist of four parts. Here we discuss it for the fasta format.

  • fasta::reader
  • fasta::writer
  • fasta::record
  • fasta::record_view

Integration CMake

Easiest way to use this repository is to clone this as a subrepo into your project, for example to lib/IVio, and then edit your CMakeLists.txt:

add_subdirectory(lib/IVio)

# ...
target_link_library(your_project ivio::ivio)

You can now happily use IVio

Code

#include <ivio/ivio.h>

int main() {
    auto reader = ivio::fasta::reader{{"somedata.fasta"}};
    for (auto record_view : reader) {
        std::cout << "id: " << record_view.id << "\n";
        std::cout << "seq: " << record_view.seq << "\n";
    }
}
